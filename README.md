## ✨ Overview

HEXACOLA.AI is an all-in-one creative AI suite that combines powerful image generation, storyboarding, scriptwriting, and AI chat assistance. Perfect for artists, writers, filmmakers, and creative professionals looking to enhance their workflow with AI-powered tools. ## 💻 Installation

1. Clone the repository:
```bash
git clone https://github.com/hexacola/hexacola-ai-creative.git
```

2. Open index.html in your web browser
3. Navigate between tools using the top navigation bar

## 📖 Usage

### Image Generation
```javascript
// Example image generation
{
  prompt: "Mystical forest scene",
  model: "flux-pro",
  style: "Cinematic"
}
```

### Storyboard Creation
1. Enter overall story scenario
2. Add frames (up to 10)
3. Generate AI scenes
4. Reorder as needed
5. Export as PDF/ZIP

### Script Writing
1. Select script type
2. Use formatting toolbar
3. Add scenes and dialogue
4. Export in desired format

### AI Chat
1. Select AI model
2. Enter your query
3. Get AI-powered responses
